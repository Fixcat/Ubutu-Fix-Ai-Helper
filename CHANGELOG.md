# История изменений FixAdm

## Версия 1.1.2 (2026-03-09)

### Исправления
- 🐛 **КРИТИЧЕСКОЕ**: Исправлена функция extract_json
  - Теперь правильно извлекает первый JSON из ответа с множественными JSON
  - Использует jq для надежного извлечения
  - Обрабатывает JSON с вложенными структурами
  - Игнорирует весь текст после первого валидного JSON

### Улучшения
- 🔧 Улучшена обработка ответов AI
  - Метод 2 использует jq -c для компактного извлечения
  - Более надежное определение границ JSON объекта
  - Fallback на несколько методов извлечения

## Версия 1.1.1 (2026-03-09)

### Исправления
- 🐛 **КРИТИЧЕСКОЕ ИСПРАВЛЕНИЕ**: Исправлена логика автоматического продолжения
  - AI теперь действительно продолжает работу автоматически после каждой команды
  - Исправлена проблема с перезаписью автоматического сообщения пользовательским вводом
  - Добавлен флаг `auto_continue` для корректной обработки автоматических сообщений
  - AI видит результаты команд и продолжает до полного завершения установки
- 🐛 **Исправлена бесконечная петля**: AI не продолжает автоматически при уточняющих вопросах
  - Определение уточняющих вопросов по ключевым словам
  - Остановка автопродолжения когда AI просит уточнить детали
  - Предотвращение зацикливания на вопросах без контекста
- 🐛 **Добавлена система повторных попыток**: AI пытается исправить ошибки до 5 раз
  - Счетчик попыток для каждой ошибки
  - Каждая попытка должна использовать ДРУГОЙ подход
  - Детальный анализ типов ошибок (permission denied, not found, port in use)
  - Автоматический сброс счетчика при успехе

### Улучшения
- 🔍 Улучшенная отладка и диагностика ошибок
  - Более детальные сообщения об ошибках API
  - Показ возможных причин проблем
  - Рекомендации по исправлению
  - DEBUG режим включается только через переменную окружения
- 🛠️ Добавлен скрипт диагностики `debug-test.sh`
  - Проверка версии и зависимостей
  - Проверка конфигурации
  - Тест API соединения для всех провайдеров
  - Детальная информация о проблемах
- 📚 Добавлено руководство по решению проблем `TROUBLESHOOTING.md`
- 🧠 Улучшен системный промпт для AI
  - Единый SYSTEM_PROMPT для всех провайдеров
  - Детальные инструкции по обработке ошибок
  - Примеры разных подходов к решению проблем
  - Анализ типов ошибок и рекомендации
  - **КРИТИЧНО**: AI должна отвечать ОДНИМ JSON за раз
  - Запрет на множественные JSON в одном ответе
  - Запрет на текст вне JSON
- 🔧 Улучшена функция extract_json
  - Извлекает ПЕРВЫЙ валидный JSON из ответа
  - Игнорирует лишний текст до и после JSON
  - Предупреждение если AI нарушает формат

## Версия 1.1 (2026-03-09)

### Новые возможности
- ✨ Добавлена команда `/settings` для изменения настроек прямо из чата
  - Смена провайдера без перезапуска
  - Обновление API ключа
  - Изменение модели
  - Полная смена всех настроек
- 🔄 Автоматическая проверка обновлений при запуске (раз в 24 часа)
- 📦 Команда `fixadm --update` для ручного обновления из командной строки
- 🔄 Команда `/update` для обновления прямо из чата
- 💾 Автоматическое создание резервной копии при обновлении
- 🔄 Автоматическая очистка истории разговора при смене настроек
- 💾 Возможность сохранить новые настройки сразу после изменения

### Улучшения
- Более удобное управление конфигурацией
- Не нужно перезапускать приложение для смены провайдера
- Показ текущих настроек (с частично скрытым API ключом)
- Детальные инструкции для AI по редактированию конфигов
- Запрет на использование интерактивных редакторов (nano, vim)
- Использование sed, echo, cat для редактирования файлов
- AI доводит установку до конца и дает инструкции по использованию
- Улучшен парсинг JSON ответов от AI (игнорирование лишнего текста)
- Строгий формат ответов AI - только чистый JSON
- AI видит результаты выполнения команд и обрабатывает ошибки
- AI не сдается при ошибках и ищет альтернативные решения
- Результаты команд передаются обратно в контекст AI
- **Автоматическое продолжение установки** - не нужно говорить "дальше"
- AI автоматически продолжает работу после каждой команды
- Остановка только при финальном сообщении с инструкциями

## Версия 1.0 (2026-03-09)

### Первый релиз
- 🤖 Поддержка 3 AI провайдеров: OpenAI, Claude, GitHub Models
- 💬 Интерактивный чат с AI
- 🧠 Память на 50 последних сообщений
- ⚡ Автоматическое выполнение команд
- 🔒 Подтверждение команд перед выполнением
- ✏️ Редактирование команд
- 💾 Сохранение конфигурации
- 📝 История команд
- 🎨 Красивый интерфейс

### Команды чата
- `/exit` - выход
- `/clear` - очистить экран
- `/save` - сохранить конфигурацию
- `/reset` - очистить историю
- `/history` - показать количество сообщений
- `/help` - справка

### Скрипты
- `fixadm.sh` - основное приложение
- `install.sh` - установка
- `uninstall.sh` - удаление
- `check-deps.sh` - проверка зависимостей
