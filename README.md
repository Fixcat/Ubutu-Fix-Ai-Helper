# FixAdm v1.1
AI-powered администратор для Ubuntu Linux

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Bash](https://img.shields.io/badge/bash-5.0+-green.svg)](https://www.gnu.org/software/bash/)
[![Ubuntu](https://img.shields.io/badge/ubuntu-20.04+-orange.svg)](https://ubuntu.com/)

Консольное приложение для администрирования Ubuntu через AI-ассистентов с поддержкой OpenAI, Claude и GitHub Models.

![FixAdm Demo](https://via.placeholder.com/800x400?text=FixAdm+Demo)

## 🚀 Возможности

- 🤖 **3 AI провайдера**: OpenAI, Claude (Anthropic), GitHub Models
- 💬 **Интерактивный чат** с AI в консоли
- 🧠 **Память на 50 сообщений** для контекста разговора
- ⚡ **Автоматическое выполнение** команд администрирования
- 🔒 **Подтверждение команд** перед выполнением
- ✏️ **Редактирование команд** перед выполнением
- 💾 **Сохранение конфигурации**
- 📝 **История команд**
- ⚙️ **Изменение настроек** прямо из чата (`/settings`)
- 🎨 **Красивый интерфейс** с цветами

## 📋 Требования

- Ubuntu 20.04+ (или другой Debian-based дистрибутив)
- bash 5.0+
- jq (для работы с JSON)
- curl (для API запросов)
- API ключ от одного из провайдеров

## 🔧 Установка

### Быстрая установка

```bash
# Клонировать репозиторий
git clone https://github.com/Fixcat/Ubutu-Fix-Ai-Helper.git
cd Ubutu-Fix-Ai-Helper

# Установить
chmod +x install.sh
sudo ./install.sh

# Запустить
fixadm
```

### Ручная установка

```bash
# Установить зависимости
sudo apt update
sudo apt install -y jq curl

# Сделать исполняемым
chmod +x fixadm.sh

# Запустить без установки
./fixadm.sh
```

## 🎮 Команды

### Команды командной строки

```bash
fixadm              # Запустить интерактивный чат
fixadm --update     # Проверить и установить обновления
fixadm --version    # Показать версию
fixadm --help       # Показать справку
```

### Команды в чате

| Команда | Описание |
|---------|----------|
| `/exit` | Выход из чата |
| `/clear` | Очистить экран |
| `/save` | Сохранить конфигурацию |
| `/reset` | Очистить память (забыть контекст) |
| `/settings` | Изменить настройки (провайдер, API, модель) |
| `/update` | Проверить и установить обновления |
| `/history` | Показать количество сообщений в памяти |
| `/help` | Справка |

## 🔐 Безопасность

⚠️ **Важно:**
- Всегда проверяйте команды перед выполнением
- API ключи хранятся в `~/.fixadm-config` с правами 600
- Можно отредактировать команду перед выполнением (опция 'e')
- Можно отменить выполнение команды (опция 'n')

Подробнее: [SECURITY.md](SECURITY.md)

## 🔄 Обновление

FixAdm автоматически проверяет наличие обновлений при запуске (раз в 24 часа).

### Ручное обновление

```bash
# Проверить и установить обновления
fixadm --update

# Или обновить вручную
cd Ubutu-Fix-Ai-Helper
git pull
sudo ./install.sh
```

### Откат к предыдущей версии

Если после обновления возникли проблемы:

```bash

sudo mv /usr/local/bin/fixadm.backup /usr/local/bin/fixadm
```

## Использование

### Запуск

```bash
fixadm
```

### Первый запуск

1. Выберите AI провайдера (1-3)
2. Введите API ключ
3. Введите ID модели
4. Сохраните конфигурацию (опционально)

### Команды в чате

- `/exit` - выход из чата
- `/clear` - очистить экран
- `/save` - сохранить конфигурацию
- `/reset` - очистить историю разговора (AI забудет контекст)
- `/history` - показать количество сообщений в памяти
- `/help` - показать справку

#### OpenAI
1. Зарегистрируйтесь на https://platform.openai.com
2. Перейдите в API Keys
3. Создайте новый ключ
4. Рекомендуемые модели: `gpt-4`, `gpt-4-turbo`, `gpt-3.5-turbo`

#### Claude (Anthropic)
1. Зарегистрируйтесь на https://console.anthropic.com
2. Получите API ключ в настройках
3. Рекомендуемые модели: `claude-3-5-sonnet-20241022`, `claude-3-opus-20240229`

#### GitHub Models
1. Получите токен на https://github.com/settings/tokens
2. Включите доступ к GitHub Models
3. Рекомендуемые модели: `gpt-4o`, `Phi-3-medium-128k-instruct`

## Память и контекст

FixAdm помнит последние 50 сообщений в разговоре, что позволяет:
- Продолжать сложные задачи
- Ссылаться на предыдущие команды
- Уточнять детали без повторения контекста

Используйте `/reset` чтобы очистить память и начать новый разговор.

## Команды в чате

- `/exit` - выход из программы
- `/clear` - очистить экран
- `/save` - сохранить текущую конфигурацию
- `/reset` - очистить историю разговора (AI забудет контекст)
- `/settings` - изменить настройки (провайдер, API ключ, модель)
- `/update` - проверить и установить обновления
- `/history` - показать количество сообщений в памяти
- `/help` - показать справку

## Изменение настроек в процессе работы

Вы можете изменить провайдера, API ключ или модель прямо из чата:

```
/settings
```

Это позволяет:
- Переключиться на другого провайдера
- Обновить API ключ
- Изменить модель
- Изменить всё сразу

При изменении настроек история разговора автоматически очищается.

## Устранение неполадок

### Ошибка: "jq: command not found"
```bash
sudo apt install jq
```

### Ошибка: "curl: command not found"
```bash
sudo apt install curl
```

### Ошибка API: "Invalid API key"
- Проверьте правильность API ключа
- Убедитесь, что у вас есть доступ к выбранной модели
- Проверьте баланс аккаунта

### Ошибка: "Permission denied"
```bash
chmod +x fixadm.sh install.sh uninstall.sh
```
