## Описание файлов

### fixadm.sh (35KB)
Основной скрипт приложения. Содержит:
- Интерфейс выбора провайдера
- Функции для работы с OpenAI, Claude, GitHub Models API
- Систему памяти (50 сообщений)
- Интерактивный чат
- Выполнение команд с подтверждением
- Сохранение/загрузка конфигурации
- Автоматическую проверку обновлений
- Функцию обновления через GitHub

### install.sh (1.2KB)
Автоматическая установка:
- Проверка прав root
- Установка зависимостей (jq, curl)
- Копирование fixadm в /usr/local/bin
- Настройка прав доступа

### uninstall.sh (3.8KB)
Полное удаление:
- Удаление исполняемого файла
- Удаление конфигурации всех пользователей
- Удаление истории команд
- Удаление истории разговоров

### check-deps.sh (1.5KB)
Проверка зависимостей:
- Проверка наличия jq
- Проверка наличия curl
- Проверка версии bash
- Инструкции по установке

### README.md (7KB)
Полная документация:
- Описание возможностей
- Инструкции по установке
- Получение API ключей
- Примеры использования
- Команды чата
- Устранение неполадок
- Информация о безопасности

### QUICKSTART.md (2.6KB)
Быстрый старт:
- Установка за 3 шага
- Основные команды
- Примеры запросов
- Ссылки на получение API ключей

### EXAMPLES.md (8.7KB)
Подробные примеры:
- Базовые операции
- Установка серверов
- Настройка безопасности
- Мониторинг
- Резервное копирование
- Диагностика проблем
- Примеры диалогов

### VERSION (0.3KB)
Информация о версии:
- Номер версии
- Дата релиза
- Список возможностей
- Список файлов

## Файлы конфигурации (создаются при использовании)

```
~/.fixadm-config          # Сохраненная конфигурация
~/.fixadm-history         # История команд
~/.fixadm-conversation    # История разговора (50 сообщений)
```

## Установленные файлы (после установки)

```
/usr/local/bin/fixadm     # Глобальная команда
```

## Размер проекта

- Общий размер: ~45KB
- Основной скрипт: 22KB

- Вспомогательные скрипты: 5KB

## Зависимости

- bash (>= 4.0)
- jq (для работы с JSON)
- curl (для API запросов)
- Ubuntu Linux (или другой Debian-based дистрибутив)
