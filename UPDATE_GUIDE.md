# Руководство по обновлению FixAdm

## Автоматическая проверка обновлений

FixAdm автоматически проверяет наличие новых версий при каждом запуске (не чаще раза в 24 часа).

### Как это работает

1. При запуске `fixadm` программа проверяет файл `~/.fixadm-last-update-check`
2. Если прошло более 24 часов с последней проверки, делается запрос к GitHub
3. Сравнивается текущая версия с версией в репозитории
4. Если доступна новая версия, вы увидите уведомление:

```
╔════════════════════════════════════════════════════════════╗
║           Доступна новая версия FixAdm!                    ║
╠════════════════════════════════════════════════════════════╣
║ Текущая версия: 1.1
║ Новая версия:    1.2
╚════════════════════════════════════════════════════════════╝

Хотите обновить сейчас? (y/n):
```

## Ручное обновление

### Способ 1: Через команду (рекомендуется)

```bash
fixadm --update
```

Это:
- Скачает последнюю версию с GitHub
- Создаст резервную копию текущей версии
- Установит обновление
- Покажет список изменений

### Способ 2: Через git

```bash
cd ~/path/to/Ubutu-Fix-Ai-Helper
git pull
sudo ./install.sh
```

### Способ 3: Полная переустановка

```bash
# Удалить старую версию
sudo ./uninstall.sh

# Скачать новую
cd ~
rm -rf Ubutu-Fix-Ai-Helper
git clone https://github.com/Fixcat/Ubutu-Fix-Ai-Helper.git
cd Ubutu-Fix-Ai-Helper

# Установить
sudo ./install.sh
```

## Резервное копирование

### Автоматическое

При обновлении через `fixadm --update` автоматически создается резервная копия:
```
/usr/local/bin/fixadm.backup
```

### Ручное

Перед обновлением можно создать резервную копию вручную:

```bash
# Копия исполняемого файла
sudo cp /usr/local/bin/fixadm /usr/local/bin/fixadm.backup

# Копия конфигурации
cp ~/.fixadm-config ~/.fixadm-config.backup
cp ~/.fixadm-history ~/.fixadm-history.backup
cp ~/.fixadm-conversation ~/.fixadm-conversation.backup
```

## Откат к предыдущей версии

### Если обновление прошло неудачно

```bash
# Восстановить из автоматической резервной копии
sudo mv /usr/local/bin/fixadm.backup /usr/local/bin/fixadm
```

### Если нужна конкретная версия

```bash
cd ~/Ubutu-Fix-Ai-Helper
git checkout v1.1  # замените на нужную версию
sudo ./install.sh
```

## Проверка версии

```bash
# Показать текущую версию
fixadm --version

# Или
fixadm -v
```

## Что сохраняется при обновлении

✅ **Сохраняется:**
- Конфигурация (`~/.fixadm-config`)
- История команд (`~/.fixadm-history`)
- История разговоров (`~/.fixadm-conversation`)

❌ **Не сохраняется:**
- Исполняемый файл (заменяется новой версией)

## Миграция между версиями

### С версии 1.0 на 1.1

Никаких дополнительных действий не требуется. Конфигурация совместима.

### Будущие версии

Если потребуется миграция, инструкции будут в CHANGELOG.md для конкретной версии.

## Отключение автоматической проверки

Если вы хотите отключить автоматическую проверку обновлений:

```bash
# Создать файл с датой в далеком будущем
echo "9999999999" > ~/.fixadm-last-update-check
```

Проверка будет отключена до 2286 года 😄

## Частые вопросы

### Как часто выходят обновления?

Обновления выходят по мере необходимости:
- Критические исправления: сразу
- Новые функции: раз в 1-2 месяца
- Мелкие улучшения: по накоплению

### Нужно ли обновляться сразу?

Рекомендуется обновляться для:
- Исправлений безопасности (критично)
- Исправлений ошибок (желательно)
- Новых функций (опционально)

### Что делать если обновление не работает?

1. Проверьте подключение к интернету
2. Убедитесь что у вас есть права sudo
3. Попробуйте обновить вручную через git
4. Создайте Issue на GitHub с описанием проблемы

### Можно ли использовать разные версии на разных машинах?

Да, но рекомендуется использовать одинаковые версии для консистентности.
