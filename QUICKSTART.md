# FixAdm - Быстрый старт

## Установка за 3 шага

```bash
# 1. Установите зависимости
sudo apt update && sudo apt install -y jq curl

# 2. Установите FixAdm
chmod +x install.sh && sudo ./install.sh

# 3. Запустите
fixadm
```

## Первый запуск

1. Выберите провайдера (1-3)
2. Введите API ключ
3. Введите модель (например: `gpt-4` или `claude-3-5-sonnet-20241022`)
4. Сохраните конфигурацию

## Основные команды в чате

| Команда | Описание |
|---------|----------|
| `/exit` | Выход из чата |
| `/clear` | Очистить экран |
| `/save` | Сохранить конфигурацию |
| `/reset` | Очистить память (забыть контекст) |
| `/settings` | Изменить настройки (провайдер, API, модель) |
| `/history` | Показать количество сообщений в памяти |
| `/help` | Справка |

## Получение API ключей

### OpenAI
- Сайт: https://platform.openai.com
- Модели: `gpt-4`, `gpt-4-turbo`, `gpt-3.5-turbo`

### Claude
- Сайт: https://console.anthropic.com
- Модели: `claude-3-5-sonnet-20241022`, `claude-3-opus-20240229`

### GitHub Models
- Сайт: https://github.com/settings/tokens
- Модели: `gpt-4o`, `Phi-3-medium-128k-instruct`
